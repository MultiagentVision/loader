# Monorepo overview

**Purpose and layout:** Same as **chessverse-monorepo**. AI Chess monorepo with pnpm workspaces: packages (frontend, backend, mobile, serverless, simulation, chat, coaching, infrastructure) and microservices (bff, train, tournament, history, videostreaming, simulation, bridge, etc.).

**Commands:** Identical. `pnpm install`, `pnpm run dev`, `pnpm run backend:start`, `pnpm run frontend:start`, `pnpm run mobile:start`, `pnpm --filter <pkg> run dev`. Package names and paths match chessverse-monorepo.

**Use:** For any detail (packages list, microservices, scripts), see [chessverse-monorepo.md](chessverse-monorepo.md).
